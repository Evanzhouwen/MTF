Member Data Documentation

bool KLDParams::debug_mode

initial value with which each bin of the joint histogram is pre-seeded

to avoid numerical instabilities due to empty or near empty bins (caused e.g. by having to assume log(0) = 0 for empty bins)

bool KLDParams::partition_of_unity

no.

of bins in the histograms used internally - dimensionality of the KLD error vector will be n_bins * n_bins; if partition_of_unity is enabled, this should be 2 more than the desired no. of bins (w.r.t normalized pixel range) since the actual range within which the pixel values are normalized is 2 less than this value to avoid boundary conditions while computing the contribution of each pixel to different bins by ensuring that pixels with the maximum and minimum values contribute to all 4 bins required by the bspl function of degree 3 used here;

double KLDParams::pre_seed

decides whether the partition of unity constraint has to be strictly observed for border bins;

if enabled, the pixel values will be normalized in the range [1, n_bins-2] so each pixel contributes to all 4 bins
